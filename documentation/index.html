Getting Started

eLearniv - React Next.js Learning Management System

Setup Development Environment

To kickstart the development of the web app with eLearniv, at first need to setup the react development environment.

Following tools are needed to setup a react dev environment:

1. Node JS

- NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site https://nodejs.org/en/

2. Install WAMP or XAMPP for MySQL/PHPMyAdmin.

Creat a database name like 'elearniv_dev' after that import the "elearniv_dev.sql" file for demo data. The elearniv_dev.sql file available in the download file. Then update the ".env" file database access credential

Cloudinary

Cloudinary is a SaaS technology company headquartered in Santa Clara, California, with offices in Israel, England, Poland, and Singapore. The company provides cloud-based image and video management services. It enables users to upload, store, manage, manipulate, and deliver images and video for websites and apps.

3. Create Cloudinary Account And Follow the below screenshots.

- https://api.cloudinary.com/v1_1/cloud-name/image/upload

- https://api.cloudinary.com/v1_1/cloud-name/video/upload

Update the cloudinary info next.config.js file

4. Create a Stripe Account.

Now from the application root directory, open the 'next.config.js' file using your code editor and update like below-

STRIPE_SECRET_KEY: "sk_test_3D..........",
STRIPE_PUBLISHABLE_KEY: "pk_test_Zy........"

Preparation

Use command line tool i.e. PowerShell or terminal and navigate to the project(elearniv-react-next) root. This is PowerShell in windows and terminal on mac/linux.

Let's have a look on the setup process:

Use terminal and navigate to the project root.

Then run : npm install

Then run : npm run dev

It'll return something like this in the console:

>elearniv-react-next@1.3.0 dev drive:\project

> next dev

ready - started on 0.0.0.0.3000, url: http://localhost:3000

[ event ] compiled successfully

Now, in the browser go to localhost:3000

For Production Build

Run npm run build to build the project. The build artifacts will be stored in the .next / directory.

Run : npm run build

See the official next.js doc here
